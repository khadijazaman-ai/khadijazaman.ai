## ✨ Features

- **Anti-FOUC Dual-State Theme System**: Seamlessly switch between Light and Dark modes. Theme state is cached in `localStorage` and integrates with system level preferences (`prefers-color-scheme`) with zero flash of unstyled content (FOUC).
- **Responsive Layout & Fluid Design**: Fully optimized across standard mobile viewports, tablet screens, up to 4K ultra-wide monitors.
- **Centered CV Button Call-to-Action**: Symmetrically positioned for visual balance and thumb-friendly ergonomics on mobile.
- **Modern Animations & Transitions**: Powered by modern CSS transitions, scroll-reveals, hover-scaling dynamics, and indigo/purple gradient accents.
- **Structured Sections**: Hero Banner, About Me, Technical Skills, Academic Projects, Interactive Timeline (Education/Achievements), and validated Contact Form.

## 🛠 Deployment Guides

### A. Deploying to GitHub Pages (Recommended)
1. In your GitHub repository, navigate to the **Settings** tab.
2. Select **Pages** from the left-hand sidebar menu.
3. Under **Build and deployment**, set the source dropdown to **Deploy from a branch**.
4. Set the branch to **`main`** and the folder to **`/ (root)`**, then click **Save**.
5. Wait 1–2 minutes, and your site will be live!

### B. Deploying to Netlify
1. Log into your Netlify dashboard and click **Add new site** > **Import from an existing project**.
2. Connect your GitHub account and select this repository (`Portfolio`).
3. Netlify will automatically parse the `netlify.toml` file. Click **Deploy Site**!
